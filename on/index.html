인턴 과제 피드백 — 온보딩 사례 조사 및 제안

1. 잘한 점

GOOD
① 단계 정의의 명확성

온보딩과 오리엔테이션의 비교표, Pre-boarding부터 Re-onboarding까지의 5단계 정의 모두 명확. 개념 정리 측면에서 좋은 출발.

GOOD
② 우수기업 선정의 다양성

국내 IT(우아한형제들·힐링페이퍼)와 해외 빅테크·금융(아마존·BoA)의 조합으로 규모·업종·문화가 다양한 사례 풀 확보.

GOOD
③ 트랙별 우선순위 표

10p에서 5개 제안을 순위·GAP·실행 트랙으로 정리.
의사결정자가 쉽게 이해할 수 있는 형식.

2. 개선이 필요한 점

FIX · 핵심
① 단계별 목표와 사례 활동의 매칭 검증 필요

2p에서 각 단계별 목표를 정의했으나, 이후 우수기업 사례 활동이 그 목표에 부합하는지 검토 필요.
검증 과정 없는 사례 나열은 정보 정리 수준에 머물게 됨

단계 정의한 목표 기업 실제 활동 매칭 검토
Pre-boarding 불안감 해소, 기대감 조성 우아한형제들 입사 가이드·일정 안내 메일 ○ 부합
아마존 사전 행정교육, 업무 프로세스 안내 △ 업무 사전 준비·생산성 가속 성격이 더 큼
힐링페이퍼 땡스 패키지(도서·상품권·편지) ○ 부합
적응(1~3개월) 초기 성과 창출, 피드백 루프 힐링페이퍼 싱크앤얼라인(히스토리·도서 공유) △ 문화 내재화 성격이 강함
아마존 AR/VR 직무 교육 △ 업무 지식 전달 성격
  • 매칭이 어긋날 경우 두 가지 해석 가능
    (a) 정의한 목표가 업계 통념과 다른 경우
    (b) 해당 기업이 그 단계에 다른 의도를 부여한 경우
  • 상기 표를 확인하면, 각 단계별 목표와 단계가 다른 경우들이 있으므로 단계 정의를 재검토 또는 재제안 해보는 것도 좋은 방향
FIX · 핵심
② "즉시 투입 가능 / 예산 투입 필요" 분류 기준

4C 카드에서 사용된 "즉시 투입 가능"이라는 표현의 의미가 불분명
예산을 말하는지(=비용 없음), 시점을 말하는지(=즉시 시행 가능) 명시 필요
10p 우선순위 표에서는 "① 예산 필요 / ② 비용 없음"으로 표기가 달라져, 두 페이지 간 기준 정합성도 점검 대상

또한 일부 제안은 분류가 실제 운영 관점에서 합리적인지 추가 검토 필요

제안 8~9p 표기 10p 표기 검토 포인트(예시)
D-5 안내 메일 표준화 즉시 투입 가능 ① 예산 필요 동일 항목에 대한 두 페이지 표기 불일치로, 통일 필요.
또한 안내 메일 표준화에 필요한 예산 내용도 작성 필요
Buddy 제도 도입 즉시 투입 가능 ② 비용 없음 형식만 도입할 경우 비용 없이 운영 가능
그러나 실제 작동을 위해서는 활동비(커피챗·식사)등 운영 비용 발생 가능성 존재
  • 위 예시는 "분류 오류"의 의미가 아닌, 분류 기준의 모호함이나 두 페이지 표기 불일치로 인해 발표 전 재점검의 필요성을 제기하는 차원
  • 분류 카테고리를 "비용 없음 / 운영비 소요(소액) / 도입비 소요(중·대)" 정도로 한 단계 세분화하면 모호함 해소 가능
FIX
③ 상상인 현황 분석의 단편성
  • 7p GAP 분석에 "OJT, 타부서 점심식사" 정도만 기재되어 현 상태 파악 어려움
  • 웰컴 키트 외 적응·Re-onboarding 단계는 "GAP 높음"만 표기, 현황란 공란. 미운영인지 미조사인지 불명확
FIX
④ 자료 점검
  • 오탈자·띄어쓰기 누락·동일 단어 및 문장 반복이 자료 전반에 산재

3. 추가로 보강하면 좋은 것

TIP
단계별 목표의 출처 명시

2p의 단계별 목표가 어떤 문헌·연구·프레임워크에서 가져온 것인지 명시 권장. 출처가 있으면 "사례가 특수한 경우", 없으면 "정의 재검토 필요"로 논의 방향이 분명해짐

TIP
상상인 현황 정밀 진단

외부 사례 비중이 크므로, 추후 실제로 관련 PJT를 진행 시에는 최근 입사자와의 인터뷰를 통해 "실제 상상인 온보딩의 개선점"을 확보하면 설득력 보강 가능
